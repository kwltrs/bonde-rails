# Bonde::Rails

Include [Bonde](https://github.com/kwltrs/bonde) in your Rails app.

## Installation

Add this line to your application's Gemfile:

    gem 'bonde-rails', github: 'kwltrs/bonde-rails'

And then execute:

    $ bundle
